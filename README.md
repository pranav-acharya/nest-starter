## Migrations
```
typeorm migration:generate -n AddEmailUsername

# To convert migrations into the dist folder into JS
$ npm run build

# Running the migration
$ typeorm migration:run

```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
